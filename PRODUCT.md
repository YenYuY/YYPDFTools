# Product

## Register

product

## Users

People who need quick PDF and image utilities in the browser, especially users who care about keeping files local instead of uploading documents to a server. They are usually completing a specific task such as merging, splitting, compressing, protecting, unlocking, rotating, or converting files.

## Product Purpose

3Y Toolbox provides privacy-first PDF and image tools that run in the browser. Success means users can identify the right tool quickly, upload files with confidence, understand processing state, recover from errors, and download the expected output without leaving the page.

## Brand Personality

Practical, trustworthy, calm. The interface should feel like a dependable workbench: quiet enough for repeated use, clear enough for first-time users, and careful with potentially sensitive documents.

## Anti-references

Avoid marketing-page hero layouts, decorative card grids, oversized gradients, unclear upload states, and novelty controls. The product should not look like a flashy landing page or a generic AI-generated SaaS template.

## Design Principles

- Put the active file task first.
- Make privacy and local processing visible but not noisy.
- Use consistent controls for every tool so repeated workflows feel familiar.
- Explain failures with recovery-focused language.
- Keep the layout stable across long filenames, translations, and mobile screens.

## Accessibility & Inclusion

Target WCAG AA contrast, visible keyboard focus, 44px touch targets, reduced-motion support, and resilient layout for Traditional Chinese, English, and Japanese text expansion.
